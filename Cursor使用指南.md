## 📦 配置概述

### 已优化的配置

| 配置项 | 设置 | 说明 |
|--------|------|------|
| AI 模型 | Claude 4 Sonnet | 云端模型，响应快 |
| 自动补全 | ✅ 开启 | Tab 键智能补全 |
| 联网搜索 | ✅ 开启 | @web 可搜索互联网 |
| Auto 模式 | ❌ 关闭 | 避免自动用贵的模型 |

### 终端工具配置

#### Claude Code (终端输入 `claude`)
- **模型**: Claude Sonnet 4 (云端)
- **权限**: acceptEdits (自动接受编辑)
- **功能**: 深度思考、代码审查

#### Codex (终端输入 `codex`)
- **模型**: GPT-5.2-codex (云端)
- **推理强度**: medium (平衡模式)
- **功能**: 智能代码生成

## 🤖 AI 功能使用

### 1. 对话模式 (Cmd + L)

**适用场景**：
- 询问代码问题
- 请求解释代码
- 讨论技术方案

**示例提问**：
```
这段代码是做什么的？请用简单的话解释

帮我写一个登录页面，要求：
1. 用户名和密码输入框
2. 记住密码功能
3. 表单验证

这个报错怎么解决？[粘贴错误信息]
```

### 2. 内联编辑 (Cmd + K)

**适用场景**：
- 修改选中的代码
- 重构函数
- 添加注释

**使用步骤**：
1. 选中要修改的代码
2. 按 `Cmd + K`
3. 输入修改指令
4. 按 Enter 应用

### 3. @ 引用功能

| 引用 | 功能 | 示例 |
|------|------|------|
| `@file` | 引用文件 | `@App.tsx 这个组件怎么用？` |
| `@folder` | 引用文件夹 | `@src/components 帮我添加新组件` |
| `@web` | 联网搜索 | `@web React 19 有什么新特性？` |
| `@docs` | 搜索文档 | `@docs React Router 怎么配置？` |
| `@codebase` | 搜索代码库 | `@codebase 用户认证在哪里？` |

### 4. Claude Code / Codex 终端使用

**启动方式**：
```bash
# 在终端输入
claude    # 启动 Claude Code
codex     # 启动 OpenAI Codex
```

**示例对话**：
```
帮我创建一个新的 React 组件叫 UserCard

分析一下这个项目的结构，给出优化建议

帮我修复 src/App.tsx 中的 TypeScript 错误
```

## 💰 省钱技巧

### 1. 选择合适的模型
- **简单问题**：用 cursor-small 或 gpt-4o-mini
- **复杂问题**：用 Claude Sonnet
- **避免**：频繁切换到 Opus/GPT-4 等高价模型

### 2. 优化提问方式
```
❌ 不好的提问：帮我写个网站
✅ 好的提问：帮我写一个用户列表组件，用 React + TypeScript，
           包含姓名、邮箱、头像三个字段，使用 Tailwind CSS 样式
```

### 3. 减少不必要的请求
- 先自己思考，再问 AI
- 一次把需求说清楚
- 使用 `@file` 引用相关代码，避免 AI 重复分析

### 4. 关闭自动索引
已经在设置中关闭了 `cursor.index.enableBackground`，
这样 Cursor 不会自动扫描整个项目消耗额度。

### 5. 查看用量
- 打开 Cursor 设置 → 订阅/Usage
- 定期检查剩余额度
- 月底前控制使用频率

## ❓ 常见问题

### Q1: AI 响应很慢怎么办？
**A**: 
1. 检查网络连接
2. 尝试切换到更快的模型（如 cursor-small）
3. 重启 Cursor

### Q2: 代码补全不出现？
**A**:
1. 确认已开启 `cursor.cpp.enabled`
2. 检查文件类型是否支持
3. 尝试多输入几个字符触发

### Q3: 终端的 claude 命令报错？
**A**:
1. 确认已安装 Claude Code 扩展
2. 检查是否已登录 Anthropic 账号
3. 运行 `claude --version` 查看是否正常

### Q4: 如何更新 Claude Code？
**A**:
在命令面板 (Cmd + Shift + P) 搜索 "Claude Code: Update extension"

### Q5: 怎么让 AI 联网搜索？
**A**:
在对话中使用 `@web` + 你的问题，例如：
```
@web 最新的 React 19 文档在哪里？
```

## 🎓 进阶技巧

### 1. 自定义 Prompt
在 `.cursorrules` 文件中定义项目级规则，AI 会遵循这些规则。

### 2. 使用 Composer (Cmd + I)
适合需要同时修改多个文件的场景，如重构、添加新功能。

### 3. Agent 模式
让 AI 自主完成复杂任务，会自动读取文件、执行命令、修改代码。

### 4. 代码审查
选中代码后右键 → "Ask Cursor" → 输入 "review this code"
