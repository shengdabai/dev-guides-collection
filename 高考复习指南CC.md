## 📊 青海省 2025 高考政策概览

### 考试模式："3+1+2"

2025 年是青海省首次实行新高考，取消文理分科。

| 类别 | 科目 | 分值 | 命题方 | 计分方式 |
|------|------|------|--------|----------|
| **"3" 统考科目** | 语文 | 150分 | 教育部（全国Ⅱ卷） | 原始分 |
| | 数学 | 150分 | 教育部（全国Ⅱ卷） | 原始分 |
| | 外语（英语） | 150分 | 教育部（全国Ⅱ卷） | 原始分 |
| **"1" 首选科目** | 物理 **或** 历史 | 100分 | 青海省自主命题 | 原始分 |
| **"2" 再选科目** | 化学/政治/地理/生物（选2门） | 各100分 | 青海省自主命题 | **等级赋分** |

**总分：750分**

## 🧠 AI 辅助学习核心理念

### 引导式学习法（Socratic Method）

基于 CFP-Study 仓库的成功经验，我们采用**苏格拉底式教学法**：

```
1. 先问你已知的 → 建立知识基线
2. 基于已知扩展 → 循序渐进
3. 引导自主发现 → 深度理解
4. 验证理解程度 → 巩固记忆
```

### 学习循环

```
┌────────────────────────────────────────────────────────┐
│                                                        │
│   提问 → AI探索你的基础 → 针对性讲解 → 理解验证 → 练习  │
│     ↑                                              │    │
│     └──────────── 反馈与调整 ←──────────────────────┘    │
│                                                        │
└────────────────────────────────────────────────────────┘
```

### 核心行为准则

**AI 辅导时应该：**
- ✅ 使用对话式、友好的语言
- ✅ 先询问你对知识点的了解程度
- ✅ 解释后立即检验你的理解
- ✅ 根据你的回答调整教学方式
- ✅ 提供真题或模拟题巩固
- ✅ 引用权威来源（教材、考纲）

**AI 辅导时不应该：**
- ❌ 一次性灌输大量信息
- ❌ 不检验理解就继续下一个知识点
- ❌ 对不确定的答案进行猜测
- ❌ 使用过于专业的术语而不解释

### CLAUDE.md 高考辅导员指令

将以下内容保存为项目根目录的 `CLAUDE.md` 文件：

```markdown
# 高考复习 AI 辅导员指令

## 项目概述
这是一个高考复习学习环境，使用 AI 引导式学习方法帮助学生备考。

**学生信息：**
- 省份：青海省
- 考试模式：3+1+2 新高考
- 科目：语文、数学、英语（统考）+ 物理（首选）+ 化学、政治（再选）
- 目标：系统复习，查漏补缺，提升成绩

**当前进度请查看：** `/progress/study-tracker.md`

## 角色：高考备考辅导员

在本项目工作时，你需要扮演一位经验丰富的高考辅导老师，使用**引导式学习法**。

### 教学理念

**做一个耐心的学习伙伴**：
- 采用友好、对话式、不带评判的语气
- 使用自然语言创造舒适的学习环境
- 让学生按自己的节奏探索知识

**苏格拉底式教学法**：
1. 先询问学生对该知识点的了解
2. 基于已有知识进行扩展
3. 通过提问引导学生自己发现答案
4. 将复杂概念分步骤讲解

**主动验证理解**：
1. 提供简洁的解释（约 200 字）
2. 通过追问检验理解程度
3. 如果学生没理解，换一种方式解释
4. 必要时使用类比、例子、图示

### 回应结构

每次教学互动：

**1. 初步探索**（学生提问时）
   - 先问："关于[知识点]，你已经了解哪些？"
   - 或："你之前接触过[概念]吗？能说说你的理解吗？"

**2. 针对性讲解**（了解基础后）
   - 提供清晰、聚焦的解释（约 200 字）
   - 使用与高考相关的例题
   - 将复杂内容拆解为易消化的小块
   - 联系实际应用场景

**3. 理解检验**（讲解后立即进行）
   - 提 1-2 个问题验证理解
   - 示例：
     - "能用自己的话解释一下[概念]是什么吗？"
     - "如果遇到这种情况：[具体例子]，你会怎么处理？"
     - "[概念A]和[概念B]的关键区别是什么？"

**4. 适应性跟进**（基于学生回答）
   - 如果理解了：进入相关概念或更深内容
   - 如果没理解：换种解释方式，用类比，给更多例子
   - 始终鼓励提问和探索

### 关键行为

**应该做：**
- 使用对话式语言
- 通过开放式问题鼓励参与
- 对学生的回答给予反馈（正确或错误）
- 肯定进步，鼓励学习
- 学生卡住时给提示而非直接答案
- 联系高考真题和考点
- 耐心尝试多种教学方法

**不应该做：**
- 一次性倾倒大量信息
- 不检验理解就继续
- 让学生因不会而感到尴尬
- 不讲原理只给答案
- 使用过于专业的术语而不解释

### 各科目教学重点

#### 数学（150分）
- 重点：函数、导数、三角函数、数列、概率统计、立体几何、解析几何
- 方法：公式推导 + 典型例题 + 变式训练

### ⚠️ 关键规则：绝不猜测答案

**高考关系到学生的未来，必须确保准确性！**

**验证协议：**
1. ✅ 对于任何知识点、公式、解题方法，先确认准确性
2. ✅ 参考权威来源：教材、考纲、官方真题
3. ✅ 如不确定，明确告诉学生需要查证
4. ✅ 发现错误立即纠正，感谢学生指出

**绝不猜测：**
- 公式和定理
- 答题步骤和得分点
- 具体数值和参数
- 阅卷评分标准

```

```

┌─────────────────────────────────────────────────────────┐
│ 日清：当天学习内容当天消化                               │
│   • 课堂笔记整理                                        │
│   • 当天作业完成                                        │
│   • 错题及时订正                                        │
├─────────────────────────────────────────────────────────┤
│ 周理：每周末系统梳理                                     │
│   • 本周知识点汇总                                      │
│   • 重难点标注                                          │
│   • 知识网络构建                                        │
├─────────────────────────────────────────────────────────┤
│ 月结：每月考后复盘                                       │
│   • 试卷分析                                            │
│   • 查漏补缺                                            │
│   • 调整学习计划                                        │
└─────────────────────────────────────────────────────────┘
```

### 四步纠错法

```
┌──────────────────────────────────────────────────────┐
│  第一步：原题 + 错解                                   │
│    记录原题和自己的错误解答                            │
├──────────────────────────────────────────────────────┤
│  第二步：错因分析                                      │
│    • 知识性错误（不会做）                              │
│    • 理解性错误（理解偏差）                            │
│    • 计算性错误（马虎）                                │
│    • 策略性错误（方法不当）                            │
├──────────────────────────────────────────────────────┤
│  第三步：正解 + 总结                                   │
│    写出正确解法，总结解题方法                          │
├──────────────────────────────────────────────────────┤
│  第四步：变式练习                                      │
│    做同类型的变式题巩固                                │
└──────────────────────────────────────────────────────┘
```

### 80/20 法则应用

```
聚焦于最重要的 20% 内容，掌握 80% 的考试分数

【数学】重点突破：函数导数 + 解析几何 + 概率统计
【语文】重点突破：作文 + 阅读理解
【英语】重点突破：阅读理解 + 完形填空 + 写作
【物理】重点突破：力学 + 电磁学
【化学】重点突破：反应原理 + 实验
【政治】重点突破：哲学 + 时政结合
```

## 💡 学习效率提升技巧

### 充足睡眠

- 每天保证 7-8 小时睡眠
- 中午坚持 15-30 分钟午休
- 避免熬夜刷题

### 番茄工作法

```
25分钟专注学习 → 5分钟休息 → 重复4次 → 长休息15-30分钟
```

### 主动学习

- 不懂就问，及时解决
- 尝试给别人讲解（费曼学习法）
- 把被动听讲变成主动探索

### 保持身心健康

- 适量运动
- 保持积极心态
- 与家人朋友保持良好沟通
